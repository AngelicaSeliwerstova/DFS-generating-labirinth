# DFS-generating-labirinth
realising depth first searching
приложение на Python с использованием pygame, которое генерирует случайный лабиринт в виде двумерной сетки и визуализирует поиск пути от стартовой клетки до финишной с помощью алгоритм DFS. Пошаговый обход клеток, отображает посещённые вершины и итоговый найденный путь, а также позволяет пользователю генерировать новый лабиринт, сбрасывать текущий поиск и запускать каждый из алгоритмов отдельно.
